📊 Retail Sales Data Analysis Project
📌 Overview

This project focuses on analyzing retail transaction data to gain insights into sales performance, customer behavior, and the impact of discounts on revenue. The goal is to perform end-to-end data analysis, starting from data cleaning and exploration to visualization and reporting, to support data-driven decision-making.

📁 Dataset

The dataset contains transactional data including:

Customer details
Product categories
Sales amount
Discounts
Order dates

The data was cleaned and transformed to ensure accuracy and consistency before analysis.

🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & EDA
SQL (PostgreSQL / MySQL / SQL Server) – Data querying
Power BI – Dashboard creation & visualization
Excel – Supporting analysis
Gamma – Presentation (PPT creation)
🔄 Steps Performed
1. Data Cleaning (Python)
Handled missing values
Removed duplicates
Performed data transformation and validation
2. Exploratory Data Analysis (EDA)
Analyzed sales trends
Identified top products and customers
Studied discount impact
3. SQL Analysis
Wrote queries to extract business insights
Performed aggregations and filtering
4. Data Visualization (Power BI)
Built interactive dashboards
Used KPIs, charts, and slicers
5. Reporting & Presentation
Created project report
Developed presentation using Gamma
📊 Dashboard

The Power BI dashboard provides:

Sales performance overview
Top customers and product categories
Discount impact analysis
Time-based sales trends

📈 Results & Insights
Identified key revenue-driving products and customers
Observed seasonal trends in sales
Analyzed the impact of discounts on sales performance
Provided insights to optimize pricing and marketing strategies
▶️ How to Run
Open Python files to review data cleaning and EDA process
Execute SQL queries in any SQL environment (PostgreSQL/MySQL/SQL Server)
Open the Power BI .pbix file to explore the dashboard
Refer to the report and presentation for detailed insights
🎯 Conclusion

This project demonstrates an end-to-end data analysis workflow, including data preprocessing, querying, visualization, and reporting. It highlights the ability to extract meaningful insights from raw data and support business decisions.
